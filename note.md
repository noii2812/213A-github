Git ជា Version Control System ប្រើប្រាស់សម្រាប់ពិនិត្យមើលការកែប្រែកូដ ឬ File របស់យើង
### Git is a version control system used to track changes of files and codes

GitHub ជា Online Storage ប្រើប្រាស់សម្រាប់ផ្ទុក Git Project
### GitHub is the online storage used to store git project

##### Git Commands #####
1. Config username and email
# git config --global user.email 'your@email.com'
# git config --global user.name 'your_username'
2. check config
# git config --list

3. Create/initialize git in project
# git init

4. Check Status
# git status

5. Add & Commit
# git add filename
# git commit -m "commit message"
Example: git add index.html
    or   git add .   (used to add all files)

*** Work Flow : Edit -> add -> commit

6. Clone Project
# git clone https://github.com/username/repo.git

7. Pull Project
# git pull
